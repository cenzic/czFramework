Demonstration of partial view and parial render

There are two demo in this section. 1. renderPartial is used when action will be called to generate model as full cycle of VMC With renderPartial there is option to refresh the view automatically. It will be render after the main view showed 2. partialView is used if action is not needed, view will be rendered directly. It will be render together with the view. It's an alias to call view engine directly $.jFormat() with some assistance in generate route. Can be call to refresh manually. Both partialView will be wrap inside a div tag.
Model info: @model.info

Demo of renderPartial

renderPartial with action only
@@renderPartial("renderPartial") renderPartial is a view and action name in this home controller

@renderPartial("renderPartial")

renderPartial with action only - refresh every 5 seconds
@@renderPartial({action:"renderPartial",refresh:5}) renderPartial is a view and action name in this home controller

@renderPartial({action:"renderPartial",refreshTime:0})

renderPartial with action only - static id
@@renderPartial({action:"renderPartial",refresh:5}) renderPartial is a view and action name in this home controller

@renderPartial({action:"renderPartial", id:"static"})

renderPartial with action and controller
@@renderPartial("renderPartial","home") renderPartial is a view with explicit controller

@renderPartial("renderPartial","home")

renderPartial with action, controller and parameter
@@renderPartial("renderPartial","home",{"name":"parameter","age":12}) renderPartial is a view with explicit controller and parameter

@renderPartial("renderPartial","home", {"name": "@model.name","age": @model.age})

Demo of partialView

partialView view
@@partialView("partialView") partialView is a view template in the context of current controller. It has empty model

@partialView("partialView",@@model)

partialView view - with id

@partialView("partialView",@@model,"partialViewId")

partialView view with model
@@partialView("partialView", {"name":"model.name", "age": 20}) partialView is a view template in the context of current controller with provided model

@partialView("partialView", {name:"@model.name", age: 20})

partialView with controller name
@@partialView("partialView","home",{"name":"BBB", "age": 20}) partialView is a view with explicit controller and model

@partialView("partialView","home", {name:"BBB", age: 20})
